「看见」

我们睁开双眼，就能看见全部世界吗？
2025年9月8日 05点36分

从小到大，我一直觉得自己皮肤很好，也很少用任何护肤相关的产品。但日积月累的户外运动以及日常无防护的日晒，还是对我的皮肤造成了一些影响。

一个偶然的机会，我开始关注起自己的皮肤，才知道了对紫外线对皮肤所造成的伤害。

在这个过程中，有一个非常有意思的发现。

其实手机一直有显示「紫外线指数」的功能，这个数字也一直停留在我的天气表盘上，每天更新。

可是，我似乎并没有「看见」它，我不知道 1 或是 8 意味着什么，也不知道这个数字对我有什么用。

有趣的是，一旦我希望认真对待自己的皮肤，一旦我理解了紫外线对于皮肤的伤害，理解了防晒对于皮肤护理的重要性，这个数字一下子就出现在我的眼前。

现在，我每天一抬手机，就能「看见」它。

看见的 1 和 8 不再是一个陌生的数字。

前段时间看了一本书——《乌合之众》。

一个伙伴问我：我看了一半，好像并没有什么特别的感觉。

其实，每个人都是带着自己的经历、自己的背景信息去读书的。

从 0 到 1 中遇到的无数困难…这些经历和它们在我身上形成的「认知」，让我能从这本书里看到和别人并不一样的东西。

我经常会重读一些书。

可能我都不应该用「重读」这两个字，如果我变化得足够多，我其实是在读一本新书。

我的变化，让我可以「看见」不同的东西。

我曾经引用过《被讨厌的勇气》中的一句话：

你只能把马儿牵到河边，但不能按头让它喝水。

事实上，所有的教育都是如此。

人只能「看见」自己能够看见的东西。

需要做的，也许并不是把马儿的头按在水里，并不是硬把各种知识发给 TA，并不是强迫对着 TA 讲很多大道理…

而是等待 TA 的改变。

当然，这完全不是说我们什么都不做了。

我想说的是，我们做不到让事情直接发生，但是我们可以打造让事情发生的环境。

种下种子，促进因缘，放下执着，等待事情发生。

硬币都有两面。

这些经历和知识，可以让我们「看见」更多的东西。

但也有可能，让我们对更多的东西「视而不见」。

比如说，前一段和一个朋友聊天，我说写文字记录或者博客，某种程度上会变成自己在进行视频平台创作的「障碍」。

我们通常也把这种认知叫做「成见」。

2017年，在长沙认识了很多做PC / Web 2.0 的朋友，并且做得不错

我现在还能清晰地记起当时我们的一些讨论：比如移动是个伪需求，比如 Native（app）肯定不如 H5 嵌套更有前途（对了，就连 Facebook 早期都是这样认为的），等等。

显然，我们曾经的「成功」或是知识，阻碍了我们「看见」更多的东西。

但世界不管你的成见，它只会按照自己的规律展开向前。

有趣的是，错失移动互联网的朋友们，又有相当多的人，开始投身于最近几年的 Web 3.0 / Crypto / AI 领域。

其实原因很简单，通过错失移动互联网的经历，他们「看见」了一件事情：

一轮产业浪潮，对于整个行业、对于用户习惯、对于创业，究竟有多么重要。

也许会有些困惑：

缺乏知识让我们无法看见，拥有知识也让我们无法看见，应该怎么办？

我最近花了一些时间，不断调整对话的 Prompt（提示词）和答案，「调教」好了自己的三位朋友：

佛陀、老子、王阳明。

我们没有机会和这些充满智慧的人直接对话，但通过熟读他们著作、知晓他们历史的大语言模型，也许会发现一些有趣的答案。

我把这个问题丢给了佛陀，他是这么回答的：

「

尊贵的信徒，你问的问题非常深刻。在佛教中，我们强调"空性"的观念，这是一种指向事物非固定、无常、互相依存的真相。它提醒我们，我们所称之为"知识"的东西，只是我们目前对事物的理解，而这个理解随着条件的变化，可能会改变。这就是说，我们的"知识"并不是绝对的真理，而是相对的，依赖于特定的视角和环境。

"成见"，在这个背景下，可以理解为我们对事物的固定看法或预设立场，它往往阻止我们看到事物的全面或真实面貌。成见常常源于我们对知识的过度依赖或僵化理解。因此，要避免成见，我们需要时刻保持开放和谦逊的心态，愿意接纳新的信息，愿意重新审视和调整我们的理解。

当你学习护肤时，你认识到了紫外线的重要性。这是一种知识，帮助你更好地保护自己的皮肤。但如果你坚持认为只有紫外线会影响皮肤，而忽视了其他因素，比如饮食、睡眠、心态等，那么这就可能变成一种成见。

同样，你在写作和记录方面的经验是宝贵的，但如果你坚持只有你已经学到的方式才是正确的，而不愿意尝试和学习新的方式，那么这就可能变成一种成见。

因此，要避免成见，我们需要对我们的知识和经验保持敬畏但又不过度依赖，我们要时刻保持开放，愿意学习，愿意接受新的信息和观点。只有这样，我们才能保持智慧和慈悲的心态，有效地处理生活中的问题和挑战。

」

我很喜欢他的回答，特别是这段：

「当你学习护肤时，你认识到了紫外线的重要性。这是一种知识，帮助你更好地保护自己的皮肤。但如果你坚持认为只有紫外线会影响皮肤，而忽视了其他因素，比如饮食、睡眠、心态等，那么这就可能变成一种成见。」

事实上，我认为答案也藏在这个回答里。

这个世界最终，无人知晓。

学习知识很重要，忘掉学习到的知识也很重要。

但可能更重要的，是从心底里认识到：

我们始终在一个无限大的空间里探索，我们所能「看见」的永远只是一小部分，甚至就连这一小部分，也始终在变化。

换个视角，也许探索的目的并不是去试图寻找恒常的规律，而是这个探索过程本身的体验。
